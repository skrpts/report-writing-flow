## Thematic Analysis Framework (Braun & Clarke)

### Phase 1: Familiarisation
- Read and re-read transcripts
- Note initial observations

### Phase 2: Initial Coding
- Generate codes systematically across dataset
- Code template:
  | Code | Data Extract | Source | Notes |
  |------|-------------|--------|-------|

### Phase 3: Theme Generation
- Collate codes into potential themes
- Create thematic map

### Phase 4: Theme Review
- Check themes against coded extracts
- Check themes against full dataset

### Phase 5: Theme Definition
- Name and define each theme
- Identify sub-themes

### Phase 6: Report Writing
- Select compelling extracts
- Relate analysis to research question
